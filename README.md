# EAMT-Statistical-Consulting
This repository contains the data and code for the EAMT Statistical Consulting project. The goal of this study is to perform a statistical evaluation of Post-Editing Machine Translation (PEMT), specifically we analyze how professional English-to-Dutch translators interact with four distinct levels of AI assistance across specialized medical and general news texts.

## 📁 Directory Structure

```text

├── README.md  
├── .gitignore  
├── data/  
│   └── data.csv                      # Original Dataset

│   └── complete-analysis-code.Rmd    # Final Code Report  
│   └── complete-analysis-code.html   
```

To run the Analysis first open the `EAMT-Statistical-Consulting.Rproj` folder.
Then make sure every required R package is installed. And finally the `analysis/complete-analysis-code.Rmd` can be ran or instead knitted into a .pdf or .html file. An output example is already present (`analysis/complete-analysis-code.html`).  


This work is purely intended for academic purposes.
